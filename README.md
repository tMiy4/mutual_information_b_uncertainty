# mutual_information_b_uncertainty
* `Iso_3Dplot.ipynb`: Calculates mutual information while varying the uncertainty of the observation model and the predicted state distribution. Generates Fig. 1.
* `karmedbandit_mmp_exp_pBS_Hqs.ipynb`: Contains the simulation code used to design the experimental conditions. Generates Fig. 2.
* `first_selection_analysis_presentation.ipynb`: Analyzes the first-trial slot choices in Conditions 2 and 3. Generates Fig. 3.
* `karmedbandit_mmp_analysis_fMRI_loop_rev.ipynb`: Calculates and records trial-by-trial differences in mutual information and uncertainty between the two slots using participants’ behavioral and observation data.
